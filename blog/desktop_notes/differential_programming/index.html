And block quotes

Aligned equations: \begin{align} \dot{x} &= Ax + Bu \tag{1.a} \\ y &= Cx + Du \tag{1.b} \end{align} Or centered equations (notice a funny way to write fractions): $$ \tag{abc} x = {-b \pm \sqrt{b^2-4ac} \over 2a}; \quad \frac{1}{2}. $$

Equations are written inline in the following way: When \(a \neq 0\), there are two solutions to \(ax^2 + bx + c = 0\).

The theorem (similarly, the definition, lemma, corolaries, ...) environment Cayley-Hamilton is written as follows. The title argument is optional [1,2,3]

If \(A\) is a given \(n \times n\) matrix and \(I_n\) is the \(n \times n\) identity matrix, then the characteristic polynomial of \(A\) is defined as \(p_A(\lambda) = \text{det}(\lambda I_n - A)\), where \(\text{det}(\cdot)\) is the determinant operation and \(\lambda\) is a variable for a scalar element of the base ring. Since the entries of the matrix \((\lambda I_n - A)\) are (linear or constant) polynomials in \(\lambda\), the determinant is also a degree-\(n\) monic polynomial in \(\lambda\), \(p_A(\lambda) = \lambda^n + c_{n-1}\lambda^{n-1} + \cdots + c_1 \lambda + c_0\).
